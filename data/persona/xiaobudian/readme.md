90后创意生活家，旅行过35+个国家和地区，考过潜水证，跳过伞，跑过马拉松，徒步过喜马拉雪山，穿越过热带雨林，挪威追过北极光，做过旅行体验师，酒店试睡师....

愿我们都能善待自己的天赋，活出生命100+种可能！
